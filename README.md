# Portfolio Website

This is a personal portfolio website project.

## Files Included

- `index.html` - Main HTML file for the portfolio
- `style.css` - CSS styling for the website
- `script.js` - JavaScript functionality
- `Godservant_Emmanuel_Resume.pdf` - Resume document

## How to Run

Simply open `index.html` in a web browser to view the portfolio website.

Alternatively, you can use a local server to serve the files:
\```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
\```

Then open your browser to `http://localhost:8000` to view the website.
